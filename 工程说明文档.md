# 六舵机 + 丝杆步进电机 + 无人机座舱 + 液位检测 + 3.5寸 TFT 触摸彩屏 综合控制系统

> **项目版本**: v1.3-TFT
> **更新日期**: 2026年7月5日
> **开发平台**: STM32F103ZE + Keil uVision5
> **芯片封装**: LQFP144（144引脚）

---

## 系统概述

本系统将原有两个 STM32F103C8T6 工程（六舵机控制工程 + 无人机座舱工程）合并迁移至 STM32F103ZE 单芯片平台，并在 v1.2 版本集成 3.5寸 TFTLCD 触摸显示屏。系统集成六路舵机PWM控制、丝杆步进电机控制、无人机座舱步进电机控制、液位检测、温湿度监控、继电器控制、TFT 彩屏看板与电阻触摸等功能，通过统一的 JSON 协议进行串口控制。

### 核心特性

- **六路舵机独立控制** — 支持0-180度精确角度控制
- **丝杆步进电机控制** — 精密线性运动控制，带上下限位保护
- **无人机座舱步进电机** — 28BYJ-48齿轮链条控制（fly命令）
- **液位检测** — ADC模拟量检测，实时百分比回报
- **双协议支持** — JSON命令 + 传统字符串命令兼容
- **实时环境监控** — SI7021温湿度传感器
- **双继电器控制** — 两路独立继电器高电平触发
- **语音控制引脚** — PB10引脚电平控制
- **舵机1自动模式** — 循环自动旋转功能
- **限位保护** — 丝杆步进电机带硬件限位开关保护
- **状态实时回报** — 自动回报角度、温湿度、液位、模式状态
- **3.5寸 TFTLCD 触摸彩屏** — HX8357DN 320×480，FSMC 16位并口驱动，开机动画 + 实时数据看板
- **XPT2046 电阻触摸** — 软件 SPI 驱动，5 点十字校准，校准数据存内部 Flash 末页（无需外置 EEPROM）

### 主循环时序规划（50ms周期，1秒循环）

| tick值 | 时间偏移 | 动作 |
|---|---|---|
| 0 | 0ms | 回报舵机1-4角度 |
| 4 | 200ms | 回报舵机5-6角度 |
| 8 | 400ms | 读取并回报温湿度（SI7021） |
| 12 | 600ms | 读取并回报液位百分比 |
| 16 | 800ms | 回报模式状态 |
| 每tick | 每50ms | 串口命令处理、舵机1自动旋转、舵机角度更新、限位LED闪烁、TFTLCD 看板差分刷新 |

系统每秒自动回报5类数据，时序错开避免串口缓冲冲突：

| tick | 回报内容 | 格式示例 |
|---|---|---|
| 0 | 舵机1-4角度 | `{"servo1":100.0,"servo2":50.0,"servo3":90.0,"servo4":90.0}` |
| 4 | 舵机5-6角度 | `{"servo5":0.0,"servo6":0.0}` |
| 8 | 温湿度数据 | `{"temp":25.36,"humi":60.12}` |
| 12 | 液位百分比 | `{"yewei":50}` |
| 16 | 模式状态 | `{"auto":1,"servo1stop":0,"switch":0,"switch1":0,"yuyin":0}` |

传感器连续失败3次后静默跳过，不再尝试读取，防止I2C挂死。

## 注意事项

### 硬件连接

1. **舵机供电**：建议使用独立5V电源，GND与主控共地
2. **丝杆步进电机**：确认驱动器细分参数（1600步/转）
3. **座舱步进电机**：28BYJ-48通过ULN2003驱动板连接PB12-15，5V供电
4. **液位传感器**：PA0引脚，输出模拟电压0-3.3V
5. **限位开关**：使用常开触点，按下时接地，接 PE0（底端）和 PE1（顶端）
6. **SI7021**：3.3V供电，需要上拉电阻

### 软件使用

1. **自动模式**：舵机1在自动模式（auto=0）下拒绝手动角度设置
2. **步进电机**：限位触发后会自动反向逃逸，属正常现象
3. **液位检测**：阈值可根据实际传感器特性调整（ADC.c中）
4. **fly命令**：执行期间为阻塞操作，电机转动完成后才能处理下一条命令
5. **JSON格式**：严格使用双引号，命令后加 `\r` 或 `\n`

### 故障排查

| 现象 | 排查方向 |
|---|---|
| 舵机不响应 | 检查PWM引脚接线和5V供电 |
| 舵机6不动 | 信号线须接 **PC6**（非旧版 PB7）；PB7 已被彩屏 FSMC 占用无法输出 |
| 丝杆步进电机不动 | 检查PA8/PB0/PB1接线，ENA低电平有效 |
| 座舱电机不转 | 检查PB12-15接线，ULN2003供电 |
| 液位检测异常 | 检查PA0连接，观察ADC原始值 |
| 温湿度不更新 | 检查PA4/PA5 I2C上拉电阻 |
| 命令无响应 | 确认波特率115200，检查JSON格式 |
| 自动模式无法手动 | 先发 `{"auto": 1}` 切换到手动模式 |
